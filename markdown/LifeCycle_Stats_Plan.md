## 1. 화면 목적

실제 수행 로그를 바탕으로 **비용 패턴**과 **실제 반복 주기**를 한눈에 보여 준다.

---

## 2. 집계 규칙

### 2.1 월별 비용

* 대상: `ActivityLog` 중 `cost`가 있고 **0보다 큰** 기록만.
* 기준일: `performedAt`의 **연-월(YYYY-MM)**.
* 표시: 최근 **12개월**까지 내림차순(월별 한 행 + 비율 막대).

### 2.2 이번 달 비용 합계

* 현재 달력 월의 `YYYY-MM`과 `performedAt` 접두 일치 + `cost > 0` 합산.

### 2.3 항목별 실제 평균 주기

* 해당 `itemId` 로그만, `performedAt` 오름차순(동일일이면 `createdAt` 보조).
* **인접한 두 기록** 사이 일수 간격을 모두 구한 뒤 **산술 평균**(소수 첫째 자리 반올림).
* 로그가 **2건 미만**이면 집계 불가 → UI에 “기록 2회 이상 필요” 표시.

### 2.4 항목별 누적 비용

* 해당 항목 로그 중 `cost > 0` 합계.

### 2.5 요약 카드

* 전체 로그 건수.
* 비용이 입력된 로그 건수.
* 이번 달 비용 합계.
